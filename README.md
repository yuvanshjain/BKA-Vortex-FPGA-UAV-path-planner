# Black-Winged-Kite-algorithm-Vortex-UAV-Path-Planner
Hardware-accelerated UAV path planner on the Zynq-7000 SoC using an Improved Black-Winged Kite Algorithm (IMBKA) with vortex-based obstacle avoidance. Implemented with Vitis HLS and PYNQ, achieving 5× faster execution than a Raspberry Pi software implementation while generating obstacle-aware trajectories in 5–10 ms.
